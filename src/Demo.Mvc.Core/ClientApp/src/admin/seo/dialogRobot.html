Le fichier robots.txt est un fichier présent sur votre site qui permet de donner des informations au différents moteurs de recherche. Lisez ce manuel robots.txt afin de mieu comprendre son fonctionnement.

Le champ "disallow" vous permet de saisir l'ensemble de url que vous ne désirez pas indexer. Par exemple écrivez dans le champ prévu à cet effet:
/page-a.html
/repertoire-a/
Après sauvegarde, cela générera dans votre fichier:
Disallow: /page-a.html/
Disallow: /repertoire-b/
