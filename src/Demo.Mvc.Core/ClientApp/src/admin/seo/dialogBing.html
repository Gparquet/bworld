Connectez-vous sur le site de Bing webmaster Tools de Microsoft.

Lorsque vous ajoutez un site, Bing a besoin de contrôler qu’il s’agit bien de votre site. Sélectionnez la méthode qui vous propose de créer une page comme ci-dessous : http://www.votredomaine.fr/BingSiteAuth.xml

Il faut alors extraire le code que vous propose Bing (exemple: «777A310D7CDD8638E147F54FAE9C4161») et copier le dans le champ prévu à cet effet. Une fois la sauvegarde effectuée, la page existera et fonctionnera sur votre site. Bing peut alors valider que c’est bien le vôtre.
